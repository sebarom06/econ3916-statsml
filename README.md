# ECON3916 - Statistical Machine Learning

This repository contains coursework, projects, and code for ECON3916: Statistical Machine Learning.

## Description

This course covers statistical and machine learning methods applied to economic data and problems. Topics include supervised and unsupervised learning, regression techniques, classification, model selection, and predictive modeling.

## Contents

- **Lectures**: Notes and materials from class sessions
- **Problem Sets**: Homework assignments and solutions
- **Projects**: Course projects and analyses
- **Code**: Python/R scripts for data analysis and modeling

## Tools & Technologies

- Python (NumPy, Pandas, Scikit-learn, Matplotlib)
- R (tidyverse, caret)
- Jupyter Notebooks
- Git/GitHub

## Getting Started
